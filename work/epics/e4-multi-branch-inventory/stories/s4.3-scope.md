# Scope — S4.3: Tariff Configuration

## In Scope

- Flyway migration: `service_tariffs` table (tariff_id UUID PK, service_id FK, branch_id FK, base_price DECIMAL(19,4), effective_from TIMESTAMPTZ, created_by, created_at)
- UNIQUE constraint on (service_id, branch_id, effective_from)
- RLS policy on `service_tariffs` filtering by `app.current_branch_id`
- `admin_reporting` grants extended to `service_tariffs`
- Permission seed: `tariff:manage`, `tariff:read`
- `ServiceTariff` entity with JPA mapping
- `TariffRepository` with active-tariff resolution query
- `TariffManagementService`: create, update (new effective-date entry), resolve active, list, search
- `TariffController` REST endpoints: POST/PUT/GET /tariffs, GET /tariffs/active, GET /tariffs/search
- Integration tests with Testcontainers (reuse S4.2 infrastructure)

## Out of Scope

- PWA tariff UI (S4.4)
- Tariff versioning beyond effective-date (no draft/publish workflow)
- Currency support (single currency assumed)
- Tariff approval workflow
- Bulk tariff import/export
- Tariff templates across branches

- [ ] `service_tariffs` table created with RLS and UNIQUE constraint
