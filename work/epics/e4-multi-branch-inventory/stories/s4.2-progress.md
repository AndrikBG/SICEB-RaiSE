## Defects Found & Fixed During T7

1. **EXPIRATION epoch day conversion** (V016 trigger): `NEW.absolute_quantity::text::date` fails for epoch day integers. Fixed in V017 migration: `'1970-01-01'::date + NEW.absolute_quantity`.

2. **@PreAuthorize `or` with throwing `@auth.check`**: SpEL `or` short-circuits on exception from first operand. Changed query endpoints to `hasAnyAuthority('inventory:read_service','inventory:read_all')`.

3. **Insufficient stock returns 500**: PG trigger raises `P0001` at commit time, wrapped as `JpaSystemException`. Added handler in `GlobalExceptionHandler` to translate to 409 with `SICEB-6002` code.

4. **Error message leaks PG context**: `rootMsg` included "Where: PL/pgSQL..." line. Sanitized to first line only.
