**Scope:** End-to-end verification with Docker Compose. Validate tariff CRUD through real HTTP calls, RLS enforcement, temporal resolution with multiple effective dates.

**Steps:**

2. Authenticate, switch to branch context
3. Create tariffs with different effective dates
4. Verify active tariff resolution returns correct one
5. Verify search by service name
6. Verify cross-branch isolation (switch branch, confirm no tariffs visible)

**Verification:**
```bash
cd apps/backend && mvn test
```

**Dependencies:** T1, T2
