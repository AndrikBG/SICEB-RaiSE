# ADD — Attribute-Driven Design documentation

This directory contains **SICEB** architecture and implementation planning aligned with the ADD method.

## Layout

| Path | Description |
|------|-------------|
| [`Design/`](Design/) | Architecture document, iteration plans, and design views (C4, diagrams). |
| [`drivers/`](drivers/) | Architectural drivers (`ArchitecturalDrivers.md`) and related inputs used by ADD. |
| [`implementation/`](implementation/) | Implementation plan, `progress.md`, ADD specs (`requeriments*.md`), Phases 1–7 guides, and [`summary/`](implementation/summary/) notes. See [`implementation/README.md`](implementation/README.md). |
